# What is this?
개발자 취업 준비를 위해 기술면접 자료를 정리해 놓은 저장소입니다. 개인적인 용도로 개설되었지만 많은 분들도 도움을 받으셨으면 좋겠습니다.
틀린 내용은 ```PR 또는 Issue```로 참여해주시면 정말 감사하겠습니다. 저장소의 주제와 관련된 좋은 자료를 추가해주시는 것도 환영입니다.
질문이나 기타 사항은 ```Issue```에 작성해주세요!
